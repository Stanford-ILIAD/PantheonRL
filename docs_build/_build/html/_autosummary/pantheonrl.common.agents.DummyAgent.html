update(reward, done)[source]

Add new rewards and done information if the agent can learn.

Each update corresponds to the most recent get_action (where record is True). If there are multiple calls to update that correspond to the same get_action, their rewards are summed up and the last done flag will be used.

Parameters:

  • reward (float) – The reward receieved from the previous action step

  • done (bool) – Whether the game is done

Return type:

None
